# Memory Index

This file is automatically loaded in every conversation. It indexes memory files stored in this directory. Keep entries concise — full content lives in the linked files.

## How to Use

- Add new memories by creating `.md` files in this directory
- Add a pointer to each new file in this index
- Types: `user` (about the developer), `feedback` (corrections/preferences), `project` (context/goals), `reference` (where to find things)
